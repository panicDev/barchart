# BarChart


##Setup
Add it in your root build.gradle at the end of repositories:
```groovy
allprojects {
        repositories {
            ...
            maven { url 'https://jitpack.io' }
        }
    }
```

## Dependency
Add the dependency
```groovy
dependencies {
    implementation "com.github.panicDev:barchart:0.0.6"
}
